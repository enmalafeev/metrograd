# Метроград

Демо мобильного 3D-симулятора поезда метро в стиле пиксель-арт. Движок — **Godot 4.7**.

Управляем поездом из кабины машиниста: разгон и торможение рычагом контроллера,
остановка у платформы, открытие/закрытие дверей. Маршрут из трёх станций:
Восточная → Центральная → Западная.

## Запуск на Mac

```
godot --path .
```
(Godot ставится через `brew install --cask godot`.)

## Управление

- **Рычаг справа** (тач/мышь): вверх — тяга, вниз — тормоз, отпустить — выбег (нейтраль).
- **Кнопка «ДВЕРИ»**: открыть/закрыть двери (доступна только при точной остановке у платформы).
- На компьютере дублируется клавишами: **W** — тяга, **S** — тормоз, **пробел** — двери.

Цель: плавно затормозить так, чтобы кабина оказалась у центра платформы,
открыть двери, подождать посадку, закрыть двери и ехать дальше.

## Структура

- `project.godot` — настройки (landscape, фильтрация Nearest для пиксель-арта).
- `scenes/main.tscn` — корень: 3D-мир в `SubViewport` (640×360) выводится на экран
  через `TextureRect` с фильтрацией Nearest → чёткая «пиксельная» картинка.
- `scripts/world.gd` — процедурная сборка тоннеля и станций из модульных сегментов.
- `scripts/train.gd` — кинематика движения (тяга/торможение/инерция) + кабина и камера.
- `scripts/main.gd` — HUD, управление, логика маршрута и остановок.
- `scripts/cab_controls.gd` — рычаг контроллера машиниста.
- `assets/textures/` — пиксельные текстуры (генерируются `tools/gen_textures.py`).

## Пересобрать текстуры

```
python3 tools/gen_textures.py
```
